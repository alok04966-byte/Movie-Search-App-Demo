# 🎬 Movie Search App

A simple web app to search movies using the [OMDb API](https://www.omdbapi.com/). Built with HTML, CSS, and JavaScript.

## 🚀 Features
- Search movies by title
- View movie details like year, rating, poster
- Responsive design

## 🛠️ Tech Stack
- HTML
- CSS
- JavaScript
- OMDb API
